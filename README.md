# 🏦 Détection de Fraude Bancaire — ML + Streamlit

## Description
Application de machine learning pour détecter les transactions
frauduleuses à partir de données bancaires.

## Modèles utilisés
- Logistic Regression
- Random Forest ✅ (modèle retenu)
- KNN
- SVM

## Technologies
- Python, scikit-learn, pandas
- Streamlit (interface web)
- joblib (sérialisation du modèle)

## Lancer l'application en local
pip install -r requirements.txt
streamlit run app.py

## Déploiement
Application déployée sur Streamlit Community Cloud :
https://fraude-detection.streamlit.app
